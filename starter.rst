เริ่มต้นใช้งาน
===============

ส่วนนี้อธิบายขั้นตอนเบื้องต้นเพื่อให้เริ่มเขียนเอกสารได้ทันที

สร้างโครงสร้างโปรเจกต์
------------------------

#. สร้างไดเรกทอรี ``_static`` สำหรับไฟล์สไตล์หรือรูปภาพเพิ่มเติม
#. สร้างไดเรกทอรี ``_templates`` หากต้องการ override แม่แบบ HTML ของ Sphinx
#. ตรวจสอบว่าไฟล์ ``conf.py`` ถูกตั้งค่าภาษาไทยและธีมเรียบร้อยแล้ว

คำสั่ง build พื้นฐาน
---------------------

เพื่อทดสอบการ build สามารถรันคำสั่ง

.. code-block:: bash

   sphinx-build -b html . _build/html

ผลลัพธ์ HTML จะอยู่ในโฟลเดอร์ ``_build/html`` และสามารถเปิดดูผ่านเบราว์เซอร์ได้ทันที
